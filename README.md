# Voice To Text Bot

Telegram bot for transcribing voice/audio messages and optionally editing the transcript with GPT.

This is a sanitized portfolio repository scaffold.

## What It Demonstrates

- Simple speech-to-text automation.
- AssemblyAI integration.
- GPT-based one-click text cleanup.
- Telegram inline-button UX.
- Docker deployment.

## Portfolio Role

This is a compact, easy-to-understand code repo and can support the larger Meeting Secretary case.
